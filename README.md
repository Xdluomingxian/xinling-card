# 心灵卡片 💡

**用经典著作的智慧，为你提供情绪价值和人生指引**

---

## 🌟 项目简介

当你情绪低落时，当你迷茫困惑时，当你得意忘形时——"心灵卡片"会从经典著作中为你寻找智慧，提供情绪价值和人生指引。

**灵感来源**：在人生低谷时阅读《毛泽东选集》，从中找到了力量。希望把这份力量传递给更多人。

---

## ✨ 核心功能

### 1. 情绪智能识别
- 自动分析用户输入的情绪状态
- 三种情绪类型：低谷鼓励、得意提醒、迷茫指引
- 置信度评估

### 2. 经典内容匹配
- 从毛泽东选集中匹配最合适的名言
- 提供原文、出处、解读、行动建议
- 后续扩展：古诗词、其他著作、歌词等

### 3. 精美卡片展示
- 渐变背景设计
- 清晰的内容结构
- 一键分享/复制

## 🚀 快速开始

### 1. 安装依赖

```bash
cd /home/ubuntu/projects/xinling-card
pip install -r requirements.txt
```

### 2. 启动后端服务

```bash
cd src
python main.py
```

服务启动后访问：http://localhost:8000

### 3. 打开前端页面

在浏览器中打开：`frontend/index.html`

或者直接访问 API 文档：http://localhost:8000/docs

## 📋 API 接口

### POST /card
生成心灵卡片

**请求示例**：
```json
{
  "user_input": "最近工作压力很大，感觉很累，想要放弃",
  "quote_source": "mao",
  "emotion_override": null
}
```

**响应示例**：
```json
{
  "success": true,
  "emotion": {
    "emotion": "低谷鼓励",
    "confidence": 0.85,
    "keywords": ["累", "放弃", "压力"],
    "description": "你正在经历一段艰难的时光..."
  },
  "card": {
    "quote_text": "星星之火，可以燎原。",
    "quote_source": "《星星之火，可以燎原》",
    "emotion_type": "低谷鼓励",
    "tags": ["希望", "坚持", "信念"],
    "interpretation": "即使现在力量微小...",
    "action": "相信自己的选择...",
    "generated_at": "2026-03-17T21:30:00"
  }
}
```

### POST /analyze
仅分析情绪（不生成卡片）

### GET /quotes
获取语录列表

### GET /emotions
获取支持的情绪类型
